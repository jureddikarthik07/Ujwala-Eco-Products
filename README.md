# Ujwala Eco Products

A responsive e-commerce website for **Ujwala Eco Products**, a small-scale manufacturer of eco-friendly jute bags based in Duvvada, Visakhapatnam.

## Features

- Responsive design
- Product catalog
- Product search
- Quick View
- Color selection
- Shopping cart
- Checkout page
- WhatsApp order integration
- Order success page
- Contact form
- Multi-language support
- SEO optimized

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Project Structure

```
index.html
cart.html
checkout.html
order-success.html

CSS/
JavaScript/
assets/
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
